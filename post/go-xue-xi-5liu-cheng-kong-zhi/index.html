go学习-5.流程控制

Posted on 2021-10-12，5 min read

[toc]

1.if-else结构

1.if 条件不需要 () 包含

# 额外支持一种临时赋值格式，作用域只在 if 循环中
if init; condition {
}
# 例
if val := 10; val > max {
}

1.错误捕获

1.所有函数的执行都有2个返回值：实际值；是否错误
当有错误时，是否错误，不是空 nil

value, err := func()
if err != nil {
    fmt.Printf("err msg")
}

2.switch

1.case 条件必须是同类型（可能是常量、变量、表达式等）;
2.case 条件常量不能重复

1.一个 case 多个条件

switch month {
    case 3, 4, 5:
        fmt.Println("春天")
    case 6, 7, 8:
        fmt.Println("夏天")
    case 9, 10, 11:
        fmt.Println("秋天")
    case 12, 1, 2:
        fmt.Println("冬天")
    default:
        fmt.Println("输入有误...")
    }

2.switch后可以不接表达式

score := 30

switch {
    case score >= 95 && score <= 100:
        fmt.Println("优秀")
    case score >= 80:
        fmt.Println("良好")
    case score >= 60:
        fmt.Println("合格")
    case score >= 0:
        fmt.Println("不合格")
    default:
        fmt.Println("输入有误...")
}

3.switch 后可接函数

switch func1(arg1) {
    case true:
        fmt.Println("该同学所有成绩都合格")
    case false:
        fmt.Println("该同学有挂科记录")
    }

4.switch 的穿透能力：fallthrough

1.fallthrough 直接执行下一个 case 语句，而不用判断条件；
2.只能穿透一层

s := "hello"
switch {
case s == "hello":
    fmt.Println("hello")
    fallthrough
case s == "xxxx":
    fmt.Println("xxxx")
case s != "world":
    fmt.Println("world")
}
# 输出
# hello
# xxxx

3.for

1.go只有 for 循环，没有其它的循环结构。没有 while 等

1.无线循环 for {}

1.for 条件可以被省略。所以无线循环使用 for {} 实现

2.for-range结构

1.可以迭代任何一个集合。类似 foreach
2.结构：for key, val := range coll {}
3.注：val 是对应索引的值的拷贝，修改，并不能影响原集合的值。如果 val 是指针，则是指针拷贝，修改后可以修改原集合的值

str := "Chinese：中文"
for key, val := range str {
    fmt.Printf("Character on position %d is: %c \n", key, val)
}

4.标签与goto

1.不建议使用：标签 和 goto；使用其它可读性更高的方式实现

5.defer 延迟语句

1.延迟声明函数时：函数最后执行；会快照函数使用的变量，不再受后续程序影响
2.如果有多个 defer，后进先出
3.defer声明会在return 后执行执行


# 1.延迟声明函数
# 函数最后执行；
# 会快照函数使用的变量，不再受后续程序影响
import "fmt"

func main() {
    name := "go"
    defer fmt.Println(name) // 输出: go

    name = "python"
    fmt.Println(name)      // 输出: python
}
# 输出 python go

# 2.多个 defer，后进先出
import "fmt"

func main() {
    name := "go"
    defer fmt.Println(name) // 输出: go

    name = "python"
    defer fmt.Println(name) // 输出: python

    name = "java"
    fmt.Println(name)
}
# 输出 java python go

# 3.defer声明会在return 后执行执行
import "fmt"

var name string = "go"

func myfunc() string {
    defer func() {
        name = "python"
    }()

    fmt.Printf("myfunc 函数里的name：%s\n", name)
    return name
}

func main() {
    myname := myfunc()
    fmt.Printf("main 函数里的name: %s\n", name)
    fmt.Println("main 函数里的myname: ", myname)
}
# 输出 go python go

1.意义

1.变量快照，避免复制变量，在上下文使用
2.多条件时，返回或者最后执行语句相同时，避免代码臃肿

# 臃肿示例
func f() {
    r := getResource()  //0，获取资源
    ......
    if ... {
        r.release()  //1，释放资源
        return
    }
    ......
    if ... {
        r.release()  //2，释放资源
        return
    }
    ......
    r.release()     //3，释放资源
    return
}
# defer。避免多次臃肿代码 r.release()
func f() {
    r := getResource()  //0，获取资源

    defer r.release()  //1，释放资源
    ......
    if ... {
        ...
        return
    }
    ......
    if ... {
        ...
        return
    }
    ......
    return
}
